# Readable
A content and comment web app using React &amp; Redux

### To Run Application

> - `git clone https://github.com/yufrances90/Readable.git`
> - `cd api-server`
> - `npm install`
> - `node server`
> - Open a new terminal 
> - `cd front-end`
> - `cd readable`
> - `npm install`
> - `yarn start`
